I. Introduction
---------------

ArchReactorOS is membership application software for your website. With this application you can deliver many kinds of membership. Nice member can manage members payment options. Nice member can manage orders and mailing lists for members. Integrated payment system (supporting PayPal).


----------------
II. Installation
----------------

Upload the files to your website. To begin installation you must execute url <%sitename%>/install/install.php. You'll be guided step by step instalation.
